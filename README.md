# Enterprise IT Technical Support & Windows Administration Guide

## 📌 Project Overview
This repository serves as a professional technical portfolio showcasing hands-on IT support tasks and system administration workflows executed within an enterprise environment (**Taif Health Cluster**). 

The documentation provides illustrated, step-by-step guides for crucial infrastructure maintenance, identity management, and system availability tasks.

> 🔒 **Security Notice:** All sensitive data, corporate computer names, network domain variations, and identifying assets have been strictly masked and anonymized to comply with enterprise data protection and compliance standards.

---

## 🛠️ Technical Workflows Demonstrated
* **Domain Integration (Active Directory):** Detailed configuration and procedures for successfully joining enterprise workstations into the official corporate network domain (`moh.gov.sa`), ensuring proper authentication and network compliance.
* **System Recovery & Password Management:** Demonstrating practical troubleshooting steps utilizing bootable recovery tools (such as Lazesoft and Windows Login Unlocker) to securely unlock user profiles, bypass access issues, and perform system password resets.
* **Workstation Standardization:** Configuring naming conventions and system properties in alignment with organizational standards.

---

## 💻 Tech Stack & Administrative Tools
* **Operating Systems:** Windows Enterprise / Windows Client Environments.
* **Directory Services:** Active Directory Domain Services (AD DS).
* **System Utilities:** Lazesoft Password Recovery, Windows Login Unlocker, Administrative Boot Menus.
